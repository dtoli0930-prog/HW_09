# HW_09
 먼저 채팅 야구게임 이기 때문에 챗인풋에 UI에 연결을 하고, 컨트롤러에 입력을 받으면 그 입력한것들이 텍스트로 출력되게 설정해주었습니다.
 
 컨트롤러에서는 내 채팅 위젯만 생성하고, 생성된 위젯에서 내가 친 채팅이라면 서버로 보내고 서버에서 모든 클라이언트에게 보이게끔 연결하였습니다.
 
 채팅 게임이기때문에 서버에만 존재해야하는 게임모드는 플레이어의 접속처리, 승리조건, 게임재시작, 게임내 맞춰야하는 숫자의 난수생성,모든플레이어의 채팅의 역할을 하게 만들었습니다.
 
 플레이어 스테이트에는 게임내의 플레이어의 간단한 정보와 플레이어가 맞췄을시 몇개를 맞추었는가를 띄우는 텍스트를 구현하였습니다.
 
게임스테이트는 게임 클리어에관한 정보와 로그인에관한정보를 전파하는 역할을 하게 만들었습니다.
